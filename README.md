# 🌐 Kaushani Rasadika — Personal Portfolio

A modern, responsive personal portfolio website built with **React** and **Vite**, featuring dark/light mode, EmailJS contact form, smooth scroll navigation, and a clean UI showcasing projects, articles, and technical skills.

---

## 🚀 Live Demo

[View Portfolio](https://kaushanirasadika.vercel.app/)

---


## 🗂️ Sections

| Section | Description |
|---|---|
| **Navbar** | Sticky navigation with dark/light mode toggle |
| **Hero** | Introduction with photo, name, role and CTA buttons |
| **About** | Personal bio and tech stack cards |
| **Projects** | Project showcase with GitHub and demo links |
| **Articles** | Medium articles section |
| **Contact** | Contact form with EmailJS integration |
| **Footer** | Social media links and copyright |
